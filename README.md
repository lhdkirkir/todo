# React Todo Application

一个使用 React 和 Tailwind CSS 构建的现代待办事项应用，采用了 Apple 风格的设计语言。

## 功能特点

- ✨ 清新简约的用户界面
- 📝 创建、编辑、删除待办事项
- ✅ 标记待办事项为已完成
- 🎨 响应式设计，适配各种屏幕尺寸
- 🌈 平滑的动画过渡效果

## 在线预览

访问 [https://lhdkirkir.github.io/todo](https://lhdkirkir.github.io/todo) 查看在线演示。

- React 18
- Tailwind CSS
- Create React App
